### **05_String_Exercise/**

### **06_Data_Structure_Exercise/**

### **07_List_Exercise/**

### **08_Dictionary_Exercise/**

### **09_Set_Exercise/**
- **Mục Tiêu:** Làm việc với tập hợp dữ liệu (set).
- **Nội Dung:**

  - Ứng dụng trong các bài toán yêu cầu quản lý tập hợp dữ liệu không trùng lặp.

### **10_Tuple_Exercise/**

### **11_Date_and_Time_Exercise/**

### **12_OOP_Exercise/**
